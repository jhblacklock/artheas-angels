Introduction

Welcome to the "Poncho" Bootstrap/HTML 5 template documentation. Thanks for buying it!

Here are some important questions (and answers) to start with:

What is "Poncho" ?: "Poncho" is a modern and lightweight commercial HTML template made for developers and webdesigners to create smaller static websites like landingpages, portfolio pages, CV sites etc. It based on HTML5/CSS3 and the responsive grid framework Bootstrap (SASS version) and comes with Bower and Gulp support. It is a toolbox with different snippets to be combined by you to a great looking (and working...) website.

What is it not: It is not a WordPress theme nor another CMS template! It is not a full and ready-made website.

What is all this Gulp, Bower, Bootstrap stuff? Do I need any advanced rocket sience degree to use this template?:
No, if you don´t have a clue what this is all about just don´t care about the /src folder in the templates ZIP archive. Just use the /template_itself folder and you are good to go. In this case you just need some basic HTML and CSS knowledge to create a website based on "Poncho". But if you are a professional frontend developer you probably love all this s*#!t. In this case take a look at the /dev folder

Installation

Its very easy to install "Poncho". You just need a bit of webspace plus a FTP tool.

Check out the HTML file locally

First of all: Extract the downloaded .zip file to your local HDD. It includes two (besides other folders and files) folders: /dist and /src. At this point: Don´t care about the /src folder.
The /dist/index.html file is your main HTML file. You can open it on your local computer. Just double click on it and your favorite browser will open "Poncho".

Upload the template to your webspace

Just upload the contens (all files and folders) from INSIDE the /dist folder via FTP to your webspace. Tatatata! Thats it...your website is online.

2.

The Contact Form

The template comes with a simple PHP mailing script and a build-in contact form. Make sure your webserver supports PHP. But than it is very easy to use:

Edit the contact.php file

Open the contact.php file located in:
src/php/contact.php
Search for the following line of code:

$to = "templatedeck@mailinator.com"; // this is your Email address
Change the E-Mail portion templatedeck@mailinator.com to your mail address, save and upload the file.

3.

Developer Stuff

Okay, I can´t give you a basic introduction to SASS, Bower and Gulp at this point. Read on if you are familiar with it. If not take some time, Google it, learn it and enjoy it! And if you don´t enjoy it: No fear, you can use this template without all this blingbling stuff. It just takes a bit more time...

Install npm, Bower, SASS and Gulp n your computer at first.
Then open you terminal and go to the: 

/src
folder inside the extracted .zip folder.

Run: 

 $ npm install
that installes all the gulp dependencies. Than type: 
$ bower install
to install all the frontend dependencies. Than everything should be there and you are ready to go.

4.

Important Gulp tasks

There are three important Gulp task to to compile the SCSS files and distribute the source files into the /dist folder:

Compile SCSS into CSS maually: 

/gulp sass

Watch file changes and compile SCSS fils into CSS automatically on save: 

/gulp watch

Clean up folders, minifies images and CSS files and move everything to the /dist folder: 

/gulp dist
